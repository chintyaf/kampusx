# kampusx
KampusX – Student Hub Space (SHS) is a digital platform designed as a central hub for students to discover and participate in various self-development activities outside formal classes. Students can explore seminars, workshops, and open classes, register for events, mark attendance, and receive e-certificates within one integrated system.

The platform also provides user profiles and a digital wallet system to simplify event payments. By combining event discovery, participation management, and certificate storage, KampusX helps students develop skills, expand their network, and prepare for future careers.
